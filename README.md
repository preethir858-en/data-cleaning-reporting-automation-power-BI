# Data Cleaning & Reporting Automation

## Project Overview
This project focuses on cleaning sales data and creating an interactive Power BI dashboard for automated reporting and business insights.

## Features
- Data cleaning and preprocessing
- Missing value handling
- Duplicate record checking
- KPI tracking
- Interactive visualizations
- Automated reporting dashboard

## Tools Used
- Power BI
- Excel / CSV
- DAX

## Dashboard Metrics
- Total Sales
- Total Orders
- Average Order Value
- Sales by Region
- Sales by Category
- Sales Target Achievement

## Project Outcome
Developed a professional reporting dashboard to analyze sales performance and improve data-driven decision-making.
